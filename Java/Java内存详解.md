##### 栈内存  

> 存储的是局部变量。凡是定义在方法中的变量为局部变量，方法外可以存在非局部变量。  
>
> 特点：  
>
> 变量所属的作用域一旦结束，该变量自动释放。  

##### 堆内存

> 存储的是数据和对象(数组也是一种对象)，即凡是new出来的都存储在堆内存中。  
>
> 特点：  
>
> - 每个实体都有一个首地址值  
> - 堆内存中的每一个变量都有默认初始值，根据类型的不同而不同  
> - 垃圾回收机制
